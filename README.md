# JavaScript Class Projects

### Overview
This repository contains my coursework and projects completed during my JavaScript class. It focuses on core JavaScript fundamentals, modern ES6+ features, DOM manipulation, and practical application development.
The goal of this class is not just to write code that works — but to understand why it works.

### Learning Objectives
Throughout this course, I focused on:
- Understanding JavaScript fundamentals (variables, functions, scope)
- Mastering ES6+ syntax (arrow functions, destructuring, modules)
- Working with arrays and objects effectively
- Applying Object-Oriented Programming (OOP) concepts
- Implementing Data Structures and basic Algorithms
- Building interactive web applications using DOM manipulation
- Writing clean, maintainable, and modular code
